# TeamOS Product Requirements Document (PRD)

## Vision
TeamOS is an AI-native team operating system focused on operational visibility, workflow execution, auditability and automated reporting.

## Problem Statement
Managers spend excessive time collecting updates. Employees spend excessive time reporting work. Existing task tools do not provide operational visibility.

## Goals
- Reduce reporting overhead.
- Increase operational visibility.
- Preserve audit history.
- Enable AI-generated reporting.

## User Roles
Admin, Manager, Employee.

## Core Modules
Authentication, Work Management, Workflow Engine, Activity Timeline, Evidence Management, AI Reporting, Dashboards.

## Success Criteria
Employee updates <60 seconds.
Manager review <5 minutes.
No manual timesheets.
Complete audit trail.

## MVP Scope
Google OAuth, Tasks, Workflows, Evidence, Dashboards, AI Summaries.
